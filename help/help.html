PenPaint user guide

Program description

PenPant is a simple graphic editor. It is similar to MyPaint.

System requirements

Windows 2000 or higher for 32-bit systems
Windows Vista or higher for 64-bit systems

Copyright information

PenPaint was made by Popov Evgeniy Alekseyevich. This program is distributed under GNU GENERAL PUBLIC LICENSE.
PenPaint uses an icon set from the Tango Desktop Project. It is in the public domain.

Program features
  • Load an image from a file or the clipboard.
  • Save the image to a file or the clipboard.
  • Allow to change the canvas size.
  • Pen drawing imitation.
  • Allow set the background and pen colors.
  • Allow to fill the boundary figure.

Set the canvas size and the pen size

Type the necessary size in the «Size» field to set the line width.
Type the necessary values in the «Width» and «Height» fields, and click on  the button to set the canvas size.
The line width and the canvas size are measured in pixels.

Set the background color and the pen color

 Click on  the button and select color from the dialog to set the pen color.
 Click on  the button and select color from the dialog to change the background color. The image will be clear when you change the background color.

About the drawing process

The cursor changes to cross when you move it inside the canvas.
Hold the left mouse button for drawing when you are moving the cursor.

Fill the boundary figure

Press the right mouse button when the cursor is placed inside the figure to fill the boundary figure.

Pick a color from the image

Hold Shift and press the right mouse button to pick a color from the image.
