PenPaint user guide

Program description

PenPant is a simple graphic editor simular to MyPaint.

System requirement

Windows 2000 or higher for 32 bit system
Windows Vista or higher for 64 bit system

Copyright information

PenPaint made by Popov Evgeniy Alekseyevich.This program distributed under GNU GENERAL PUBLIC LICENSE.
PenPaint uses icon set from Tango Desktop Project. It is a public domain.

Program features
  • Load image from file or clipboard
  • Save image to file or clipboard
  • Allow change canvas size
  • Pen drawing imitation
  • Allow set background and pen colors
  • Allow fill boundary figure

Set canvas size and pen size

Type necessary size in «Size» field to set line width.
Type necessary values in «Width» and «Height» fields and click on  button to set canvas size.
Line width and canvas size measure in pixels.

Set background color and pen color

 Click on  button and select color from dialog to set pen color.
 Click on  button and select color from dialog to change background color. Image will be clear when you change background color.

About drawing process

Cursor changes to cross when you move it inside canvas.
Hold left mouse button for drawing when you are moving cursor.

Fill boundary figure

Press right mouse button when cursor disposes inside the figure to fill boundary figure.

Pick color from image

Hold Shift and press right mouse button to pick color from image.
